
Aplicativo para Windows<br>
Plataforma: Desktop (Windows).<br>
Banco de Dados: Utiliza MySQL como base de dados, conectando-se a um servidor de banco de dados centralizado.<br>
Funcionalidade Principal: Cadastro e gerenciamento de atletas, onde o usuário pode inserir dados como nome, idade, categoria e outras informações. Todos os dados ficam armazenados no banco MySQL.
Manipulação de Turmas: Permite organizar atletas em turmas com base em critérios específicos. O usuário pode criar, editar e gerenciar as turmas e os dados ficam centralizados no servidor MySQL, possibilitando a integração de dados entre diferentes usuários ou dispositivos.
Aplicativo WEB<br>
Plataforma: Web, desenvolvido com a biblioteca Flet (Python).<br>
Banco de Dados: Também utiliza MySQL, conectando-se ao mesmo servidor de banco de dados centralizado que o aplicativo Windows.<br>
Funcionalidade Principal: Cadastro de atletas com as mesmas informações (nome, idade, etc.), com os dados armazenados no servidor MySQL.<br>
Manipulação de Turmas: O aplicativo Flet permite criar e gerenciar turmas de atletas. A interface facilita a visualização em tempo real dos atletas em cada turma, permitindo a adição e remoção rápida, com os dados sincronizados no banco MySQL.
